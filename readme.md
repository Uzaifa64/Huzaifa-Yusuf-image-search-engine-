# Image Search Engine

A fast, modern, and responsive **Image Search Engine** that lets users find high-quality images using the **Unsplash API**. This project demonstrates how to integrate third-party APIs, handle dynamic search queries, and display results in a clean and interactive UI.

## ‚ö° How It Works

1. User enters a search keyword in the input field.
2. The app sends a request to the Unsplash API with the keyword.
3. The API returns image data (URL, author, description).
4. The app dynamically displays images in a grid format.
5. Users can click images to view full size or download.
